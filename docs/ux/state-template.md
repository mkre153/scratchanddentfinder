# Page: State Page

**Example:** `/scratch-and-dent-appliances/arizona/`

## Goal (1 sentence)
Confirm relevance for the state, explain scratch-and-dent locally, and route users to cities or stores.

## Primary User Question
"Is this relevant where I live, and how does scratch & dent work in my state?"

---

## Required Sections (STRICT ORDER)

### 1. State Hero
**Purpose:** Local relevance + value

**Must include:**
- State-specific headline
- One-sentence description
- Primary CTA: Browse Stores
- Secondary CTA: How Scratch & Dent Works

---

### 2. Trust Strip
Same bullets as homepage trust strip.

---

### 3. Short Explainer
**Purpose:** Contextual education

**Must include:**
- 2–3 sentences explaining scratch & dent
- Brief mention of warranties (high level)

**Constraints:**
- No bullet lists
- No long paragraphs

---

### 4. City Navigation
**Purpose:** Narrow intent geographically

**Must include:**
- Top cities as links or cards

**Constraints:**
- Must appear before full store listings
- No pagination in this section

---

### 5. Store Listings (Statewide)
**Purpose:** Browse available options

**Each store card must include:**
- Store name
- City
- Appliance categories (if available)
- Primary action (Call / View)

---

### 6. State Buying Guide
**Purpose:** SEO + trust

**Must include:**
- One short paragraph (4–5 lines max)
- State-specific context (outlets, warehouses, turnover)

---

### 7. Soft CTA
**Purpose:** Contribution or alerts

**Options:**
- Suggest a store
- Get updates for this state

---

## DO NOT
- Add filters above listings
- Add long SEO blocks
- Change URL structure
- Rank or badge stores

- [ ] City links resolve correctly

- [ ] Internal links unchanged
