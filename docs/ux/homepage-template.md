# Page: Homepage

## Goal (1 sentence)
Establish trust, clearly explain what ScratchAndDentFinder does, and route users to their state.

## Primary User Question
"Is this site useful and trustworthy for finding discounted appliances near me?"

---

## Required Sections (STRICT ORDER)

### 1. Hero
**Purpose:** Outcome-first clarity

**Must include:**
- Outcome-based headline (saving money)
- One-sentence explanation of what the site does
- Primary CTA: Browse Stores by State
- Secondary CTA: How It Works

**Constraints:**
- No search input
- No filters
- No long explanations

---

### 2. Trust Strip
**Purpose:** Reduce skepticism immediately

**Must include (exactly 4 bullets):**
- Real local stores
- No paid rankings
- Updated regularly
- Free to use

**Constraints:**
- Icons + short text only
- No testimonials
- No numbers or stats here

---

### 3. How It Works
**Purpose:** Light education without friction

**Must include exactly 3 steps:**
1. Minor cosmetic damage
2. Big discounts (20–60% off)
3. Fully functional appliances

**Constraints:**
- Each step ≤ 1 sentence
- No warranty deep-dive
- No store-level detail

---

### 4. Browse by State
**Purpose:** Route users deeper into the funnel

**Must include:**
- Grid or list of state links
- Links go to `/scratch-and-dent-appliances/[state]/`

**Constraints:**
- No alphabetical dump longer than viewport
- Prefer card/grid layout

---

### 5. Why Use Scratch & Dent Finder
**Purpose:** Reinforce value proposition

**Must include:**
- 3–4 bullets focused on outcomes (not features)
- Examples:
  - Avoid paying full retail prices
  - Find local outlets fast
  - Compare multiple stores in one place

---

### 6. Soft CTA
**Purpose:** Optional engagement without pressure

**Must include one:**
- Get updates
- Suggest a store

**Constraints:**
- No popups
- No email gate above the fold

---

## DO NOT
- Add filters
- Add SEO text blocks
- Add testimonials
- Add ads or promotions

---

## Acceptance Criteria (Gate)
- [ ] Page purpose is clear in <5 seconds
- [ ] CTA routes only to state pages
- [ ] URLs, metadata, sitemap unchanged
- [ ] LCP / CLS unchanged or improved
