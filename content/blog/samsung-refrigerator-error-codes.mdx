## Samsung Refrigerator Error Codes

Diagnose your Samsung refrigerator with this complete error code guide.

## Common Error Codes

| Code | Meaning | Solution |
|------|---------|----------|
| **1E/5E/SE** | Fridge defrost sensor | Check sensor, may need replacement |
| **2E** | Freezer defrost sensor | Check/replace sensor |
| **4E** | Defrost heater | Check heater and thermostat |
| **5E** | Fridge defrost sensor | Replace sensor |
| **6E** | Ambient sensor | Check sensor connection |
| **8E** | Ice maker sensor | Check ice maker sensor |
| **21E** | Freezer fan error | Check fan motor |
| **22E** | Fridge fan error | Check fan motor, clear obstructions |
| **41E** | Freezer sensor | Replace thermistor |
| **84E** | Compressor lock | Reset power, may need service |
| **85E** | Compressor error | Contact Samsung service |
| **OF/OFF** | Demo mode | Press and hold Energy Saver + Lighting for 3 seconds |
| **PC ER** | Communication error | Reset power |

## How to Reset Samsung Refrigerator

**Standard Reset:**
1. Press **Power Freeze** + **Power Cool** simultaneously
2. Hold for 8-10 seconds
3. Wait for chime
4. Release buttons

**Power Reset:**
1. Unplug refrigerator
2. Wait 5 minutes
3. Plug back in

**Exit Demo/Shop Mode:**
1. Press **Energy Saver** + **Lighting** (or **Power Freeze** + **Fridge**) for 3 seconds
2. Display should return to normal

## Samsung Ice Maker Reset

If ice maker shows errors:
1. Press and hold **Ice Type** button for 10 seconds
2. Ice maker will run through test cycle
3. Wait 24 hours for normal ice production

- Compressor is not running
