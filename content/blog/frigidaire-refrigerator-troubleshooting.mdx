## Frigidaire Refrigerator Troubleshooting

### Not Cooling

1. **Check temperature settings** — Fridge should be 37°F, freezer 0°F
2. **Clean condenser coils** — Located underneath or behind
3. **Check door seals** — Replace if cracked or loose
4. **Clear vents** — Don't block with food
5. **Listen for compressor** — Should run periodically

### Ice Maker Not Working

1. **Check water supply** — Ensure valve is open
2. **Verify ice maker is ON** — Check arm or switch position
3. **Check freezer temperature** — Must be 0°F or below
4. **Inspect water filter** — Replace if older than 6 months
5. **Clear ice jams** — Remove stuck cubes

### Common Error Codes

| Code | Meaning |
|------|---------|
| SY EF | Evaporator fan failure |
| SY CF | Condenser fan failure |
| SY CE | Communication error |
| H1 | Freezer sensor issue |
| HI | High temperature alert |
