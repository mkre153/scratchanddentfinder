## Whirlpool Washer Error Codes

Diagnose your Whirlpool washing machine with this complete error code guide.

## Most Common Codes

| Code | Meaning | Quick Fix |
|------|---------|-----------|
| **F21/F02** | Long drain | Check drain hose, clean pump filter |
| **F01** | Primary control failure | Reset power, may need control board |
| **F05** | Water temperature sensor | Check sensor connection |
| **F06** | Motor tachometer | Check motor connections |
| **F07** | Motor control unit | Reset, may need MCU replacement |
| **F09** | Overflow | Check inlet valve, pressure switch |
| **F20** | No water detected | Check water supply, inlet valves |
| **F22** | Door lock issue | Clean door latch, check lock mechanism |
| **F23** | Heater failure | Check heating element |
| **F24** | Water temperature sensor | Replace sensor |
| **F26** | Door switch error | Check door switch |
| **F27** | Overflow | Check pressure switch |
| **LOC/LC** | Control locked | Hold Control Lock 3 seconds |
| **Sud/Sd** | Excess suds | Use HE detergent, reduce amount |

## How to Reset Whirlpool Washer

**Method 1: Power Reset**
1. Unplug washer from outlet
2. Wait 60 seconds
3. Plug back in
4. Open and close door 6 times within 12 seconds

**Method 2: Button Reset**
1. Press **Power** button
2. Select any cycle
3. Press **Power** button again
4. Washer should reset

## Clean the Drain Pump Filter

Most F21/F02 errors are caused by a clogged filter:

1. Locate filter access door (front bottom)
2. Place towel and shallow pan underneath
3. Turn filter cap counterclockwise slowly
4. Remove debris (coins, hair, small items)
5. Replace filter and secure cap
