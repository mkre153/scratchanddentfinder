## GE Dishwasher Error Codes

Use this guide to diagnose and fix your GE dishwasher. Find your error code below for the cause and solution.

## Common Error Codes

| Code | Meaning | Solution |
|------|---------|----------|
| **C1** | Drain issue | Check drain hose, filter, and garbage disposal |
| **C2** | Drain timeout | Clear blockage, check drain pump |
| **C3** | Drain motor issue | May need drain pump replacement |
| **C4** | Fill problem | Check water supply valve is open |
| **C5** | Fill timeout | Check inlet valve and water pressure |
| **C6** | Water temperature issue | Check water heater, may need heating element |
| **C7** | Turbidity sensor | Clean sensor or replace |
| **C8** | Detergent dispenser | Check dispenser mechanism |
| **PrS** | Pressure sensor | Service required |
| **888** | Display test mode | Press Start to exit |

## How to Reset GE Dishwasher

**Method 1: Button Reset**
1. Press and hold **Start/Reset** for 3 seconds
2. Wait for cycle to cancel
3. Close door and start new cycle

**Method 2: Power Reset**
1. Turn off dishwasher at circuit breaker
2. Wait 60 seconds
3. Turn power back on
4. Start new cycle
