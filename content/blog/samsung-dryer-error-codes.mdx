## Samsung Dryer Error Codes

| Code | Meaning | Solution |
|------|---------|----------|
| **HE/hE/HE1** | Heating error | Check element, thermal fuse |
| **tS/tO** | Temperature sensor | Replace thermistor |
| **dF/dC** | Door open | Close door, check latch |
| **FE** | Power frequency error | Check power supply |
| **bE** | Button stuck | Check control panel |
| **9E1** | Power supply error | Check outlet, may need service |
| **9C1** | Communication error | Reset power |
| **Clg** | Vent blockage | Clean vent, check exhaust |
| **HC** | Gas ignition fail | Check igniter, gas supply |
| **Et** | Key error | Check control board |
| **FC** | Filter clogged | Clean lint filter |

## How to Reset Samsung Dryer

1. Press **Power** to turn off
2. Unplug from outlet
3. Wait 1-5 minutes
4. Plug back in
5. Turn on and test

## Samsung Dryer Not Heating?

Check these in order:
1. **Lint filter** — Clean before each load
2. **Vent hose** — Clear any blockages
3. **Thermal fuse** — May need replacement
4. **Heating element** — Test with multimeter
