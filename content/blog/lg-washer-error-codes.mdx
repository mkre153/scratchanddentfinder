## LG Washer Error Codes

| Code | Meaning | Solution |
|------|---------|----------|
| **OE** | Drain error | Check drain hose, clean filter |
| **UE/UB** | Unbalanced load | Redistribute clothes evenly |
| **LE** | Motor locked | Reset power, check for obstructions |
| **DE** | Door open | Close door firmly, check latch |
| **FE** | Water overflow | Check inlet valves |
| **PE** | Water level sensor | Check sensor, may need replacement |
| **CE** | Current error | Power surge, reset breaker |
| **PF** | Power failure | Restart cycle |
| **CL** | Child lock | Hold Child Lock 3 seconds |
| **tCL** | Tub clean needed | Run Tub Clean cycle |
| **dE1/dE2** | Door error | Check door switch |
| **E6** | Clutch error | Service required |

## How to Reset LG Washer

1. Press **Power** to turn off
2. Unplug from outlet
3. Wait 60 seconds
4. Plug back in
5. Turn on and run cycle

## Clean the Drain Filter (OE Fix)

1. Open filter access door (front bottom)
2. Place towel and container below
3. Pull out drain hose, remove cap, drain water
4. Turn filter counterclockwise
5. Clean debris
6. Replace and secure
